# 20260215-003457 new-api 流水线改为官方镜像转推部署

## 变更目标
- 按 UI Frontend 风格重写 Jenkins Pipeline。
- 不在 Jenkins 本地构建镜像，直接使用官方 calciumion/new-api:latest。
- 在一个 Pod 中部署应用（无 GPU）。

## 修改文件
- jenkins-deploy/pipeline-new-api-single-pod.groovy

## 主要调整
1. 删除源码 checkout + docker build 阶段。
2. 增加官方镜像拉取并转推 Harbor：
   - pull: docker.io/calciumion/new-api:latest
   - push: harbor.internal.quchiai.com/sailfish/new-api:latest
3. 使用 Helm ase-app 安装，资源设为通算 CPU/内存。
4. 保留 3000 端口服务。
5. 挂载 /data 用于默认 SQLite 数据持久化。

## 说明
- 当前配置不包含 Redis/Postgres，依赖 new-api 默认 SQLite 模式。
- 如后续需要高并发/生产数据库，可再补充外部 DB 与 SQL_DSN 配置。
