# 20260215-000544 Jenkins 71g-3 取消 sidecar 部署（单卡运行）

## 变更目的
- 当前实例只有一张 GPU，sidecar 与主容器竞争 GPU 资源。
- 调整为仅部署主容器，不再部署 llama.cpp sidecar。

## 修改文件
- jenkins-deploy/pipeline-deploy.md

## 具体修改
1. 删除 sidecar 镜像配置与同步
- 删除 IMAGE_REPO_SIDECAR / IMAGE_TAG_SIDECAR
- 删除 sourceSidecarImage / 	argetSidecarImage
- 删除 docker pull/tag/push sidecar 镜像步骤

2. 删除 Helm sidecar 相关参数
- 删除 ports[5]=llamacpp:18080
- 删除 sidecars[0] 整段注入参数
- 删除环境变量 GLM5_SIDECAR_URL

3. 删除部署后 sidecar 自动 patch 逻辑
- 删除 kubectl patch deployment 注入 llama-cpp-glm5 的整个 shell 块

## 结果
- 部署仅包含主容器（ComfyUI/Ollama/OpenCode Manager）。
- 不再创建 sidecar 容器，不再占用额外 GPU 资源。
