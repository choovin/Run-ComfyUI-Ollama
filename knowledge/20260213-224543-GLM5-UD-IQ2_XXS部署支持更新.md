# 20260213-224543 GLM5-UD-IQ2_XXS部署支持更新

## 目标
在本项目中增加对 Unsloth `GLM-5-GGUF:UD-IQ2_XXS` 的一键部署支持。

## 代码变更
- `start.sh`
  - 新增环境变量开关：`DEPLOY_GLM5_UD_IQ2_XXS`
  - 当开关为 `1/true/yes` 且未设置 `OLLAMA_MODEL1` 时，自动注入：
    - `hf.co/unsloth/GLM-5-GGUF:UD-IQ2_XXS`
  - 若用户已设置 `OLLAMA_MODEL1`，保持用户配置不覆盖。
- `README.md`
  - 在环境变量表增加 `DEPLOY_GLM5_UD_IQ2_XXS`
  - 补充 GLM-5 UD-IQ2_XXS 的部署示例与说明
  - 增加参考链接：`https://unsloth.ai/docs/models/glm-5`

## 使用方式
在实例环境变量中添加：

```bash
DEPLOY_GLM5_UD_IQ2_XXS=true
```

如需手动指定模型，也可设置：

```bash
OLLAMA_MODEL1=hf.co/unsloth/GLM-5-GGUF:UD-IQ2_XXS
```

## 备注
该更新仅增加项目内自动注入逻辑，不影响已有 `OLLAMA_MODEL[1-6]` 下载机制。
